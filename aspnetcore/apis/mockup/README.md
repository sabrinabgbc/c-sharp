This project was developed to practice structuring an api, using non-persistent data, which comes from arrays created in the controller, simulating an user api.

- The api was built with ASP.NET Core.

- There is no database connection or manipulation.

- Was not building front end.

- The api return data in json format.



# Resume
Mockup api without database manipulation to simulating a users api.

## Methods variables

* list users

* list user by id

* register user

* delete

* update

# Project architecture

* Framework: ASP.NET Core
